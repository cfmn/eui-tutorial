Container and Layouts

Any UI in this library consists of containers that are nested in each other.

Containers are located in widget package.

1import "github.com/ebitenui/ebitenui/widget"

There is a single container type which can be created like this.

1c := widget.NewContainer()

You can compose containers in each other. The order of addition does not matter.

1shirt := widget.NewContainer()
2shelf := widget.NewContainer()
3store := widget.NewContainer()
4
5shelf.AddChild(shirt)
6store.AddChild(shelf)

The container have several options to setup.

1c.SetPadding(10)
2c.SetBorder(10)
3c.SetBorderRadius(10)
4c.SetBackgroundColor("#000000")
